Evento API

GET /api/events

Parameters

  • country

    (int) id of country - get events in this country.

  • city

    (int) id of city - get events in this city.

  • venue

    (int) id of venue - get events in this venue.

  • category

    (int) id of category - get events in this category.

  • date

    (string) date in format YYYY-mm-dd - get events starting this date.

  • limit

    (int) number of elements returned per page

  • page

    (int) page number
